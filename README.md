🎧 AI Music Recommendation Web Application

A Machine Learning–based web application that recommends songs based on user input using similarity techniques and presents results through an interactive Flask web interface.

🚀 Features

🎵 Smart song recommendation system
🔍 Search with typo handling
🎤 Voice search support
🤖 Machine Learning–based similarity model
🌐 Flask-powered web application
🎨 Modern UI with animations
⚡ Fast and responsive results

🛠️ Technologies Used

Python
Machine Learning
Flask
HTML, CSS, JavaScript
Pandas, NumPy
Scikit-learn
SpeechRecognition API

⚙️ How It Works

User enters a song name in the web interface
Input is matched using similarity algorithms
Machine Learning model finds similar songs
Recommended songs are displayed instantly
Voice input can also be used for searching

▶️ How to Run the Project

1️⃣ Clone the repository

git clone https://github.com/DEEPENRRASINGH2/Music_Recommendation.git
cd Music_Recommendation

2️⃣ Install dependencies

pip install -r requirements.txt

3️⃣ Run the Flask app

python app.py

4️⃣ Open in browser

http://127.0.0.1:5000/
